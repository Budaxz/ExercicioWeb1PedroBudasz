# Perfil Geek - Homem-Aranha 🕷️

Uma página web estática e responsiva criada para exibir o perfil de um personagem geek. O objetivo deste projeto é aplicar os fundamentos do desenvolvimento web front-end sem o uso de frameworks, focando na manipulação do DOM e estruturação semântica.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização temática, uso de variáveis de cor (CSS Custom Properties) e Flexbox.
* **JavaScript (Vanilla):** Manipulação da árvore do DOM para injeção dinâmica das habilidades do personagem.

## 💻 Como executar o projeto

Como é um projeto puramente estático (Vanilla), não é necessário instalar nenhum pacote ou rodar um servidor local.

1. Clone este repositório para a sua máquina:
   ```bash
   git clone <URL_DO_SEU_REPOSITORIO>
Abra a pasta perfil-geek.
```

Dê um duplo clique no arquivo index.html para abri-lo diretamente no seu navegador de preferência.
